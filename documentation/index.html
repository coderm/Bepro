      How-to...?

      How to start using Taurus template?

      In dowloaded package you will find three folders: blank, html and lesscss. Lesscss is version of template with LessCss, html is a ready to use compiled version, and blank is cleaned version with only one html page and without sampels.

      How to use /lesscss version:

      1. Upload files from downloaded package tou your webserver.
      2. Remove from /js folder files: charts.js, maps.js, demo.js(smaples), demo-editable.js
      3. Remove from /img samples folder. Replace logo with your own.
      4. Open css/styles.less find .page-container.page-boxed and set width for fixed version of site(by default 1200px)
      5. Use blank.html as basic HTML file

      How to use /html version:

      It's same as lesscss but there alredy compiled less file and included in all pages.

      How to use custom progress bar?

      There two actions for progress bar show - create new progressbar with options, update - update progressbar with options, destroy - remove.

          $.mpb(action,options); // call functions
    
    $.mpb('show',{value: [0,10]}); // Call default progress bar with 10% progress
    $.mpb('update',{value: [10,100]}); // Update progress bar to 100% progress
    $.mpb('destroy'); // Remove progress bar

      Options:

      value: array/integer. If isset array first element of it should be integer value from, second to. If isset integer it will be value to

      position: NULL/string. Available position: bottom. Default is NULL(top).

      speed: integer. Set speed of animation. Default: 20;

      state: NULL/string. There 4 available custom states: success,warning,info,danger. Default NULL - base theme color.

      complete: NULL/function. Callback function calling after animation finished.

       

      You can find this function in js/plugins.js
